배경 및 변경 내용

🔍 변경 이전의 문제

  • 기업이 공고 등록 시 직종을 직접 선택하는 구조로, 직종 오선택 빈번
  • 자사 직종 체계의 구조적 한계(하이어라키 불명확, 직무·직업·산업 개념 혼재)로 포괄적 직종 다수 선택 경향 발생
  • 결과적으로 매칭 대상은 넓어지나, 기업이 원하지 않는 직군의 인재 수락이 발생 → 기업 만족도 저하

🚀 변경 내용 (2026.04.01 적용)

  • 공고 내용(텍스트)을 LLM으로 분석하여 직종을 자동 추출
  • 추출된 직종 중 Score 70% 이상인 직종만을 매칭에 활용 (실험 기반 임계값)
  • LLM 추출 직종이 없는 경우(이미지 공고 등)는 기존 방식으로 fallback 처리
  • 직종 체계 개편 작업과 병행 진행 중 → 현 직종 체계 한계로 효과가 일부 희석될 수 있음

✅ 해석 — 과도한 직종 선택 제거

LLM 추출 결과, 기존 방식에서 존재하던 6개 이상 직종 선택 공고가 완전히 사라졌습니다. 기업이 어뷰징 또는 직종 혼재 이슈로 선택하던 과포괄 직종이 LLM에 의해 자동 정제된 효과로 해석됩니다. 직종 수 평균도 3.37 → 2.92개로 감소, 매칭 정밀도가 이론상 향상되었습니다.

⚠ 한계 — 매칭 모수 감소 가능성

직종 수 감소는 매칭 대상 구직자 모수 축소를 의미합니다. 4월 1~3회차 기준 발송 건수가 3월 대비 증가한 것은 공고 수 자체가 늘어난 영향(372→488개)이며, 공고당 평균 발송 수는 오히려 708건 → 623건으로 감소했습니다. 이는 직종 수 정제로 인한 모수 감소 신호일 수 있어 지속 모니터링이 필요합니다.

✅ 긍정 신호 — 수락률 상승

발송 대비 수락 전환율이 0.46% → 0.52%로 약 13% 개선되었습니다. 공고 수 증가로 절대 발송 수가 늘었음에도 수락률이 높아진 것은, LLM이 추출한 직종이 구직자에게 더 적합하게 제안되고 있음을 시사하는 긍정적 신호입니다.

🚨 경계 신호 — 열람률 하락 (원인 불명확)

전체 열람률이 7.93% → 6.71%로 약 15.4% 하락했습니다. 현재 가능한 원인으로 최소 두 가지를 고려해야 합니다. 첫째, 발송 후 경과 시간 차이입니다. 4월 공고는 등록한 지 얼마 되지 않아 응답이 아직 누적되지 않았을 가능성이 있습니다(3월 공고는 동기간 이후에도 열람이 누적됨). 둘째, 4월 공고 자체의 품질 차이입니다. 직종 추출과 무관하게 4월에 등록된 공고의 내용·조건·기업 매력도가 3월보다 낮을 경우 열람률이 떨어질 수 있습니다. 이 두 가지는 현재 데이터만으로 분리할 수 없어, 열람률 하락을 LLM 변경의 부작용으로 단정하기 어렵습니다.

주석: 수락 수 절대값은 공고 수 증가(+31.2%)의 영향을 받습니다. 공고당 평균 수락 수(3.27→3.21건)는 거의 변화가 없으므로, 수락률 지표를 핵심 KPI로 사용하는 것이 적절합니다.

⚠ 중요 발견 — 1~3회차 수락률 역설적 하락, 열람률 해석 주의

전체 수락률은 올랐으나, 1~3회차(초기 반응) 수락률은 모두 하락했습니다(1회차: 2.89%→2.45%, 3회차: 2.67%→1.88%). 1~3회차 열람률도 42.01% → 26.81%로 크게 하락했습니다. 다만 이 열람률 하락은 두 가지 해석이 가능합니다. ① 4월 공고가 등록 직후라 아직 응답이 충분히 누적되지 않았을 가능성, ② 4월 공고 품질(내용·조건)이 상대적으로 낮아 구직자 관심 자체가 낮았을 가능성입니다. 이를 분리하려면 동일 공고에 대한 시간 경과 후 누적 열람 데이터가 필요하며, 현 시점에서 LLM 변경의 직접 영향으로 단정하기 어렵습니다.

구조적 특이점: 5회차에 전체 발송 건의 약 70%가 집중되어 있으나 수락률은 0.03~0.05%에 불과합니다. 5회차 발송의 효율성 자체를 재검토할 여지가 있습니다.

✅ LLM 적용 공고의 우세한 성과

같은 4월 데이터 내에서 LLM 적용 공고(Y)의 수락률 0.54%가 fallback 공고(N)의 0.34%보다 약 59% 높습니다. 열람률도 6.98% vs 4.52%로 LLM 적용 공고의 초기 반응이 유의미하게 높습니다. 동일 기간·동일 환경 하에서의 A/B 비교에 가장 가까운 데이터로, LLM 적용 효과를 지지하는 핵심 근거입니다.

⚠ 해석 시 주의사항

  • fallback(N) 공고는 주로 이미지 공고로 추정되며, 텍스트 기반 공고와 공고 품질 자체가 다를 수 있습니다
  • 공고 유형 차이(이미지 vs 텍스트)가 수락률 차이에 개입할 수 있어, 순수 LLM 효과만을 분리하기 어렵습니다
  • LLM=N 샘플(39개 공고)이 상대적으로 적어 통계적 안정성에 한계가 있습니다

결론 및 향후 과제

📌 종합 판단 — "조건부 유효"

전체 수락률(0.46%→0.52%)과 LLM 적용 공고의 수락률 우위(0.54% vs 0.34%)는 이번 변경이 구직자-공고 간 매칭 정밀도를 높이는 방향으로 작동하고 있음을 시사합니다. 다만 1~3회차 초기 반응률 하락, 열람률 전반 하락, 짧은 분석 기간(17일), 공고 수 증가라는 외부 변수 존재 등을 감안할 때 현 시점에서 완전한 효과 검증은 어렵습니다. 지속 모니터링과 구조적 개선이 병행되어야 합니다.

긍정적 신호

  • 전체 수락률 13% 개선 (0.46%→0.52%)
  • 6개 이상 직종 과선택 공고 완전 제거
  • LLM 적용 공고의 수락률·열람률이 fallback 공고 대비 유의미하게 높음
  • 직종 수 양방향 정제(감소 40.3% + 증가 35.6%) — 단순 모수 감소가 아닌 정밀화
  • LLM 적용률 92.0%로 안정적인 시스템 작동

우려 신호 / 미확인 사항

  • 1~3회차 수락률 일제 하락 (2.89%→2.45%, 2.67%→1.88%)
  • 전체 열람률 15.4% 하락 — 단, 원인 불명확(공고 등록 후 경과 시간 부족 또는 4월 공고 품질 차이 가능성 모두 존재)
  • 공고당 평균 발송 수 12% 감소 → 모수 축소 가능성
  • 5회차 발송 비율 과다(70%)에 비해 수락률 극히 낮음(0.03%)

분석 한계

  • 분석 기간 17일 — 통계적 신뢰도 제한
  • 공고 수 증가(+31.2%) 등 외부 변수 통제 불가
  • 기업 만족도 측정 수단 미비 (설문 준비 중)
  • LLM fallback 원인 미분리 (이미지 공고 vs 직종 추출 실패)
  • 매칭 모수(실시간) 데이터 미추출

향후 과제

  • [단기] 5월 말 기준 동기간 비교 재분석으로 신뢰도 제고
  • [단기] 기업 만족도 설문 조기 시행 → 수락 품질 측정
  • [단기] 열람률 하락 원인 파악 — 시간 경과 후 누적 열람 추이 및 3월·4월 공고 품질 차이(업종·기업규모 등) 비교 분석 필요
  • [중기] 직종 체계 개편 완료 후 LLM 추출 정확도 재측정
  • [중기] 5회차 발송 효율성 재검토
  • [중기] LLM fallback 원인 분리 추적 체계 구축
보고 권고: 현 데이터를 근거로 "성공"을 확정하기보다는, 긍정적 초기 신호를 바탕으로 지속 모니터링 및 보완 개선을 병행하겠다는 방향으로 경영진 보고를 구성하는 것이 데이터에 충실한 접근입니다.
